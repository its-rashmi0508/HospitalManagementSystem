# Hospital Management System

A web-based Hospital Management System developed using Java Servlet, JSP, JDBC, MySQL, Bootstrap, Maven, and Apache Tomcat.

## Features

* Add Patient
* Patient Registration Form
* Doctor Management
* Appointment Management
* MySQL Database Connectivity
* Responsive Bootstrap Dashboard

## Technologies Used

* Java
* JSP
* Servlet
* JDBC
* MySQL
* Bootstrap 5
* Apache Tomcat 10
* Maven

## Project Structure

HospitalManagementSystem
├── src
├── pom.xml
├── database.sql
├── README.md
└── screenshots

## How to Run

1. Install MySQL.
2. Execute database.sql.
3. Configure database credentials in DBConnection.java.
4. Build using Maven.
5. Deploy WAR file in Tomcat.
6. Open:

http://localhost:9090/HospitalManagementSystem
